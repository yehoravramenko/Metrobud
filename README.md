
# Metrobud

Source code of Metrobud game.
Aura Software (c) 2023-2024

### For developers

Metrobud uses its own custom build system implemented in Bash scripting language to provide robust and fast compilation and linking.

    ./build.sh

***Avaliable commands***

To build a release version:

    ./build.sh
To build a debug version:

    ./build.sh debug
To build glad.dll (normal GLAD but in the library):

    ./build.sh glad
